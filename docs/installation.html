Installation

Get started with Akindutire Authorization in your Laravel application.

Requirements

  • PHP 8.1 or higher
  • Laravel 9.0 or higher
  • Composer

Install via Composer

Install the package using Composer:

composer require akindutire/authorization

Publish Configuration

Publish the configuration file to customize the package settings:

php artisan vendor:publish --tag=akindutire-authorization-config

This will create a configuration file at config/akindutire-authorization.php.

Database Setup

Generate Migration

Create a migration to add permission columns to your model's table:

php artisan make:permission-migration users

This generates a migration file that adds two columns:

  • allowed_permissions - JSON array of granted permissions
  • revoked_permissions - JSON array of explicitly denied permissions

Run Migrations

Apply the migration to your database:

php artisan migrate

Custom Column Names

If you need different column names, you can customize them in the migration or configure them in config/akindutire-authorization.php:

'column_names' => [
    'allowed_permissions' => 'permissions',
    'revoked_permissions' => 'denied_permissions',
],

Add Trait to Model

Add the HasPermissions trait to any Eloquent model that needs authorization:

use Akindutire\Authorization\Traits\HasPermissions;

class User extends Authenticatable
{
    use HasPermissions;

    // Your model code...
}

Warm Reflection Cache (Optional)

For production environments, pre-warm the reflection cache during deployment:

php artisan permission:cache

This eliminates cache misses on first requests and improves initial response times.

Auto-Invalidation: When auto-invalidation is enabled in config, caches automatically update when attributes change. No manual cache clearing needed during development.

Verify Installation

Test that everything is working by checking permissions in tinker:

php artisan tinker

$user = User::first();
$user->grantPermission(['can_view', 'can_edit']);
$user->hasPermission('can_view'); // true
